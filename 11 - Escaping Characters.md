## Characters You Can Escape


| Character	Name | Name                |
|:--------------:|:-------------------:|
| \              | backslash           |
| `              | backtick            |  
| *              | asterisk            |  
| _              | underscore          |  
| { }            | curly braces        |  
| [ ]            | brackets            |  
| ( )            | parentheses         |  
| #              | pound sign          |
| +              | plus sign           |
| -              | minus sign (hyphen) |
| .              | dot                 |
| !              | exclamation mark    |
| \|             | pipe                |
